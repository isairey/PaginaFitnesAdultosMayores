# 🧓💪 Fitness Senior

Página web enfocada en el **bienestar físico de adultos mayores**, desarrollada con tecnologías web básicas para ofrecer rutinas seguras y accesibles.

---

## 🚀 Descripción

**Fitness Senior** es una aplicación web que promueve la actividad física en personas mayores mediante ejercicios adaptados, guías visuales y seguimiento básico del progreso.

El sitio está diseñado para ser **ligero, fácil de usar y accesible**, permitiendo su uso en cualquier dispositivo sin necesidad de instalaciones.

---

## 🎯 Objetivos

- Fomentar el ejercicio en adultos mayores  
- Mejorar la salud y movilidad  
- Ofrecer rutinas seguras y fáciles de seguir  
- Facilitar el acceso desde cualquier navegador  

---

## ✨ Características

### 🏋️ Rutinas de Ejercicio
- Ejercicios de bajo impacto  
- Guías paso a paso  
- Rutinas organizadas por nivel  

### 📊 Seguimiento Básico
- Registro simple de actividades  
- Progreso diario (local)  

### 📱 Diseño Responsive
- Adaptable a celular, tablet y PC  
- Interfaz clara y accesible  

---

## 🧱 Tecnologías utilizadas

- **HTML5** → Estructura del sitio  
- **CSS3** → Estilos y diseño responsive  
- **JavaScript (Vanilla)** → Interactividad
